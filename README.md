## 🎯 Konteks & Problem Statement

> **Potensi zakat nasional Indonesia mencapai Rp327 triliun, namun baru Rp41 triliun yang terhimpun — gap sebesar 87%.**
> — *BAZNAS & UMSIDA (Februari 2026)*

| Fakta | Data |
| :--- | :--- |
| 💰 Potensi Zakat Nasional | **Rp327 Triliun** |
| 📉 Realisasi Terhimpun | **Rp41 Triliun** (12.5%) |
| 🕳️ Gap yang Hilang | **Rp286 Triliun** |
| 👥 Muzakki Potensial | **192 Juta Jiwa** |
| 🕌 Zakat Fitrah 2026 | **Rp50.000/orang** (BAZNAS) |

### Akar Masalah:
1. **Kurangnya literasi perhitungan zakat** — Banyak muzakki tidak tahu cara menghitung kewajiban zakat untuk berbagai jenis aset (maal, penghasilan, emas, pertanian, perdagangan)
2. **Rendahnya kepercayaan pada lembaga** — Muzakki lebih memilih menyalurkan langsung karena kurangnya transparansi dan akuntabilitas lembaga pengelola zakat
3. **Proses pembayaran yang terpisah-pisah** — Tidak ada platform yang mengintegrasikan hitung zakat + bayar langsung + lacak donasi dalam satu alur seamless

### Solusi ZakatKu:
Satu platform terintegrasi yang menjembatani muzakki dengan mustahik secara **akurat, transparan, dan mudah** — didukung AI untuk perhitungan dan Mayar untuk pembayaran.

## ✨ Fitur Utama

### 1. 🧮 AI Kalkulator Zakat (5 Jenis)
Kalkulator cerdas yang menghitung kewajiban zakat secara akurat berdasarkan ketentuan syariah Islam dan nisab terkini. Mendukung perhitungan real-time dengan validasi nisab otomatis.

| Jenis Zakat | Nisab | Tarif | Input |
| :--- | :--- | :--- | :--- |
| **Zakat Maal** | 85g emas (≈ Rp85.000.000) | 2.5% | Tabungan, deposito, investasi, properti, hutang |
| **Zakat Penghasilan** | Rp6.650.000/bulan | 2.5% | Gaji, bonus, pendapatan lain |
| **Zakat Emas & Perak** | 85 gram emas | 2.5% | Berat emas (gram), harga per gram |
| **Zakat Pertanian** | 653 kg gabah | 5% / 10% | Hasil panen (kg), jenis irigasi |
| **Zakat Perdagangan** | 85g emas (≈ Rp85.000.000) | 2.5% | Aset dagang, piutang, hutang dagang |

### 2. 💝 Program Donasi Transparan
6 kampanye donasi terkurasi dengan progress tracking real-time:
- 🎓 **Beasiswa Yatim Ramadhan** — Pendidikan 200 anak yatim se-Indonesia
- 👁️ **Operasi Katarak Gratis** — Bantu 500 lansia penderita katarak
- 🏪 **Modal UMKM Mustahik** — Modal usaha untuk 100 mustahik
- 🌊 **Bantuan Bencana Banjir** — Tanggap darurat banjir Kalimantan Selatan
- 🕌 **Pembangunan Masjid Dhuafa** — Masjid layak di 5 desa terpencil
- 📦 **Paket Sembako Ramadhan** — 10.000 paket sembako keluarga kurang mampu

### 3. 📊 Dashboard Transparansi
Dashboard publik yang menampilkan data real-time: total terhimpun, jumlah donatur, progress setiap program, dan riwayat donasi terbaru — membangun kepercayaan muzakki melalui akuntabilitas penuh.

### 4. 💳 Integrasi Pembayaran Mayar
Checkout seamless via **Mayar Payment Gateway** — mendukung transfer bank, virtual account, QRIS, dan dompet digital. Invoice otomatis dan webhook untuk konfirmasi pembayaran real-time.

### 5. 🌗 Dark / Light Mode
Desain bertema Islamic (emerald green + gold accent) dengan dukungan mode gelap dan terang. Auto-detect preferensi sistem pengguna.

### 6. 📱 Responsive Design
Tampilan optimal di desktop maupun perangkat mobile berkat **Tailwind CSS** dan komponen **shadcn/ui**.

## 🚀 Panduan Instalasi

### Prasyarat
- **Node.js** v18+ ([download](https://nodejs.org/))
- **npm** v9+

### 1. Clone Repositori
```bash
git clone https://github.com/GhazyUrbayani/zakatku.git
cd zakatku
```

### 2. Instal Dependensi
```bash
npm install
```

### 3. Jalankan Development Server
```bash
npm run dev
```
Server berjalan di `http://localhost:5000`. Frontend dan backend otomatis di-serve pada port yang sama via Vite.

### 4. Build untuk Production
```bash
npm run build
NODE_ENV=production node dist/index.cjs
```

## 💳 Integrasi Mayar Payment

Saat ini ZakatKu menggunakan **mock payment URL** untuk demo. Untuk mengaktifkan pembayaran nyata via Mayar:

### Langkah 1: Daftar Akun Mayar
1. Buka [mayar.id](https://mayar.id) → klik **"Daftar Sekarang"**
2. Isi data bisnis/organisasi, verifikasi email
3. Login ke Mayar Dashboard

### Langkah 2: Dapatkan API Key
1. Di dashboard, buka **Integration** → **API Keys**
2. Klik **"Generate New API Key"**
3. Simpan API key (hanya ditampilkan sekali!)

### Langkah 3: Konfigurasi di Kode
Buka file `server/routes.ts`, cari bagian `POST /api/donations`, dan uncomment kode Mayar API call:

```typescript
// Uncomment dan ganti YOUR_API_KEY:
const response = await fetch("https://api.mayar.id/izar/v1/invoice/create", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.MAYAR_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    name: donorName,
    email: donorEmail,
    amount: amount,
    description: `Donasi ${type} - ${campaign.title}`
  })
});
```

### Langkah 4: Set Environment Variable
```bash
export MAYAR_API_KEY="your-mayar-api-key-here"
npm run dev
```

### Langkah 5: Setup Webhook (Opsional)
1. Di Mayar Dashboard → **Integration** → **Webhooks**
2. Tambah webhook URL: `https://your-domain.com/api/mayar/webhook`
3. Pilih event: `payment.success`
4. Donasi akan otomatis ter-update statusnya menjadi "paid" saat pembayaran berhasil

### Referensi Dokumentasi Mayar
- [Mayar MCP Server](https://mcp.mayar.id/) — 16 tools, 7 kategori (Account, Invoice, Customer, Transaction, Membership, Unpaid, Product)
- [Mayar API Documentation](https://api.mayar.id/docs/)

## 🎬 Proses Vibecoding

> **Submisi untuk [Mayar Vibecoding Competition Ramadhan 2026](https://mayar.id)**
> 
> Tema: *"Code the Vibe, Let Agentic AI Do the Rest!"*

Seluruh proses development ZakatKu dilakukan **100% melalui vibecoding** dengan **Perplexity Computer** (Claude Sonnet):

### 1. 🔍 Riset & Ideasi (AI-Driven Research)
Meminta AI untuk meneliti *"masalah terbesar di Indonesia yang bisa diselesaikan dengan AI + pembayaran digital."* AI melakukan web search real-time dan menemukan **gap zakat Rp286 triliun** sebagai problem statement terkuat — relevan dengan Ramadhan, berdampak sosial besar, dan ideal untuk integrasi Mayar.

### 2. 🎨 Arsitektur & Design System
AI merancang:
- Arsitektur fullstack (React + Express + TypeScript + Tailwind + shadcn/ui)
- Data model (campaigns, donations, zakatCalculations) dengan Drizzle + Zod
- Art direction bertema Islamic: **emerald green** (#0E8A5F) + **gold** (#D4910A) + geometric patterns
- Responsive layout dengan dark/light mode

### 3. 🏗️ Build (Pure Vibecoding — Zero Manual Coding)
Seluruh kode ditulis oleh AI melalui satu sesi prompt:
- Schema database + Zod validation
- 9 API endpoints (campaigns, donations, zakat calculator, stats, webhook)
- 5 halaman frontend dengan komponen reusable
- Kalkulator zakat 5 jenis dengan logika nisab & tarif syariah
- Integrasi Mayar invoice creation API (mock + template real)
- 30+ shadcn/ui components
- Custom Islamic theme dengan CSS variables

### 4. 🧪 QA & Deploy
AI melakukan quality assurance menggunakan **Playwright** browser automation:
- Screenshot desktop (1280px) dan mobile (375px)
- Verifikasi dark mode toggle
- Testing interaksi kalkulator
- Build production → Deploy ke live URL

### 5. 📦 Git & Repository
AI membuat GitHub repo, commit, dan push kode — semua dari satu sesi vibecoding.

**⏱️ Total waktu: ~30 menit dari nol sampai live.**
